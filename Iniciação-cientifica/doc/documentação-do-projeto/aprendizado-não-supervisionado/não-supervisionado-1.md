🧠 Resumo — Aprendizado Não Supervisionado
O aprendizado não supervisionado é uma técnica de machine learning voltada para descobrir padrões ocultos e estruturas nos dados sem usar rótulos pré-definidos. Em vez de prever algo, ele explora, segmenta e reduz os dados, buscando entendimento profundo em grandes volumes de informação.

⚙️ Principais Procedimentos:
- Agrupamento (Clusterização): Divide dados em grupos com padrões semelhantes — útil para segmentação de clientes ou análise comportamental.
- Mineração de Regras de Associação: Identifica relações frequentes entre itens — como em carrinhos de compras ou interações entre medicamentos.
- Redução de Dimensionalidade: Simplifica os dados mantendo o essencial — ideal para análise de imagens médicas ou dados complexos de alta dimensão.

🛠 Exemplos Práticos:
- Segmentação de clientes em bancos ou e-commerce.
- Recomendação de conteúdo em plataformas de streaming.
- Diagnóstico em saúde com análise de imagens.
- Estudo de interações químicas em medicamentos.

🧮 Fundamento Teórico:
- Baseado em dados não rotulados.
- Focado na estrutura interna dos dados.
- Usa tabelas onde linhas = observações e colunas = atributos.

🧭 Aplicações e Benefícios:
- Exploração de dados sem conhecimento prévio.
- Organização e compressão de grandes bases de dados.
- Personalização de serviços com base em comportamentos.

🧑‍🚀 Dica extra:
O Kaggle é uma ótima plataforma para explorar conjuntos de dados e praticar esse tipo de aprendizado com casos reais.
